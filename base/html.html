# 自定义HTML标签，随心造

前置知识 (opens new window)

TIP

和vue的自定义指令含义差不多，触类旁通嘛

vue自定义指令 (opens new window)

类似像uni-app那些自定义标签就是这么来的


class MyDialog extends HTMLElement {
  constructor() {
    super()
  }
  
  connectedCallback() {
    // 首次被插入文档时调用
  }
  
  disconnectedCallback() {
    // dom被删除时调用
  }
  
  adoptedCallback() {
		// dom被移动到新文档时调用
  }
  
  attributeChangedCallback() {
		// 自身属性被增删改查时调用 
  }
}
window.customElements.define('my-dialog', MyDialog)

// 如果你想指定特定的 dom 类型，可通过下段代码找到所有的衍生
{
  let res = []
  for (let key of Object.getOwnPropertyNames(window)) {
    let item = window[key]
    if (item && item.prototype instanceof HTMLElement) {
      res.push(key)
    }
  }
  console.log(res)
}
